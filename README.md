# Project Starter

A setup starter

## Stack

- **Frontend** — Astro + Vue.js
- **CMS** — Sanity.io
- **AI** — (your AI backend)

## Structure

```
project-root/
├── frontend/       # Astro + Vue app
├── cms/            # Sanity Studio
└── .gitignore
```

## Getting Started

**Frontend**

```bash
cd frontend/your-astro-project
npm install
npm run dev
```

**CMS**

```bash
cd cms
npm install
npm run dev
```

## Environment Variables

Create a `.env` file in the frontend folder:

```
SANITY_PROJECT_ID=your_project_id
SANITY_DATASET=production
```
